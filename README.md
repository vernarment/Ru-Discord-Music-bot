# Ru-Discord-Music-bot
Самопальный музыкальный бот, оптимизированный через нейронку, сделан на русском. РАБОТАЮТ ТОЛЬКО ЮТУБ ССЫЛКИ

Команды

!join
|
!leave
|
!play <youtube_url>
|
!pause
|
!resume
|
!skip
|
!volume <0-100>
|
!loop off / track / queue
 
  
Установите на свою машину ffmpeg для работы
